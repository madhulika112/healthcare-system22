About my Healthcare System

This project is a smart healthcare platform designed to help users understand their health using simple inputs like symptoms, food images, and body measurements. It uses logical models and AI-based concepts to generate useful health insights.

🚀 Features

🩺 Health Checkup

Analyze symptoms and predict possible diseases with risk level and suggestions.

🍎 Diet Analyzer

Upload food images and get nutrition details like protein, carbs, fiber, and calories.

⚖️ Body Check (BMI)

Check your BMI using height and weight and get personalized suggestions.

⚙️ How It Works

  • User enters data (symptoms, food, body details)
  • System analyzes inputs using logic
  • Predictions and suggestions are generated
  • User gets easy-to-understand results

💻 Technologies Used

  • Python (Flask)
  • HTML, CSS
  • Basic AI Logic

🎯 Purpose

The main goal of this project is to make health awareness simple and accessible. It helps users take early steps towards better health by understanding their body.
